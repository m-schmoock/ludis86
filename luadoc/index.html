Module ludis86

Udis86 C disassembler bindings for Lua.

“Udis86 is an easy-to-use, minimalistic disassembler library (libudis86) for the x86 class of instruction set architectures.”

This bindings were former part of Lua code injection framework: mmBBQ

The GitHub project site is: here There you can find sources and checkout links. You can also just do:

git clone https://github.com/willsteel/ludis86.git

Overview

The project contains a standard Makefile that is used to build shared Lua libraries for Windows and Linux. The contained C code part of the bindings can as well be integrated or hard-linked into a Lua projekt.

QuickUsage

-- initialize and get the first disassembled instruction as string
local ludis86 = require("ludis86")
ud = ludis86.init_addr_intel32(0x401000)
ud:dis()
ud:asm()

-- disassemble and print 10 instructions
for i = 1, 10 do
    print(string.format("+%04X %016s %016s", ud:off(), ud:hex(), ud:asm()))
end
